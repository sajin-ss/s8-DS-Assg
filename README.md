# s8-DS-Assg
Multicast socket in Java

Usage:

Compile:

$javac MulticastServer.java 

$javac MulticastClient1.java 

$javac MulticastClient2.java 


Open 3 terminals

Terminal 1: 
------------
$ java MulticastClient1


Terminal 2: 
------------
$ java MulticastClient2


Terminal 3: 
------------
$ java MulticastServer
