# s8-DS-Assg
Multicast socket Java

Usage:

Compile:

$javac MulticastSocketServer.java 

$javac MulticastSocketClient1.java 

$javac MulticastSocketClient2.java 


Open 3 terminals

Terminal 1: 
------------
$ java MulticastSocketClient1


Terminal 2: 
------------
$ java MulticastSocketClient2


Terminal 3: 
------------
$ java MulticastSocketServer
